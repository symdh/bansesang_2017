Project Year : 2017 <br/>
Project name : 웹 커뮤니티 사이트 (개인프로젝트) <br/>
Environment : Apache / MySQL / PHP / Linux <br/>
Language : PHP 7 / HTML / JavaScript(Jquery) / CSS <br/>

특징
- Framework 적용 (Codeigniter 3)
- 디자인 패턴 : MVC
- API 적용 (Daumeditor, HTMLPurifier, reCAPCHA)
